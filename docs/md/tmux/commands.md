## Shell Prompt

  - `tmux rename-session -t [current-name] [new-name]` = Rename session
  - `tmux list-session` = List current sessions

## Tmux Command Prompt

<kbd>Prefix</kbd>, <kbd>:</kbd> = Tmux command prompt

  - `rename-session -t [current-name] [new-name]` = Rename session
