## Git Integration

**`git diff` replacement**

  - `ydiff -s`


**`git log` replacement**

  - `ydiff -ls`


**Diff Fit to screen**

  - `ydiff -w 0 -s`


**Pipeline usage with `diff`**

  - `diff -u a b | ydiff -w 0 -s`
