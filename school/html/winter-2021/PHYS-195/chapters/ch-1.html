Chapter 1: Measurements

SI Units

My Notes

When cancelling out units with different powers, match the smaller unit to the bigger one

For example. Suppose we want to cancel out ft^{3} with \frac{n}{ft}:

\left ( \frac{3^{3}ft^{3}}{1^{3}yd^{3}} \right ) \left ( \frac{12in}{1ft} \right ) \to \left ( \frac{3^{3}ft^{3}}{1^{3}yd^{3}} \right ) \left ( \frac{12^{3}in^{3}}{1^{3}ft^{3}} \right )

then we can cancel out the ft^{3}.
